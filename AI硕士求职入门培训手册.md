## 第一课：先搞懂计算机产业的"食物链"

### 1.1 一个比喻：造汽车

想象你要造一辆汽车：

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  发动机 (Engine)        →    芯片 (CPU/GPU)              │
│  你踩油门，它转动              你写代码，它计算            │
│                                                         │
│  变速箱 (Transmission)  →    操作系统 (OS)              │
│  把动力传给轮子               把计算结果传给软件          │
│                                                         │
│  仪表盘 (Dashboard)     →    应用软件 (App)             │
│  你看到的界面                 你用的微信、抖音            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**核心观点**：芯片是"发动机"，没有它都干不了！

### 1.2 谁是老大？

全世界计算机产业的"食物链顶端"：

| 位置 | 谁在这 | 为什么牛 |
|------|--------|----------|
| 🥇 芯片公司 | Intel, AMD, Nvidia | 造"发动机"的，卡别人脖子 |
| 🥈 操作系统 | 微软Windows, 苹果iOS, Google Android | 造"变速箱"的，控制软件生态 |
| 🥉 应用软件 | 抖音、微信、ChatGPT | 造"仪表盘"的，离用户最近 |

**你是做算法的**，在"应用软件"这一层。但现在为什么要关心芯片？

### 1.3 为什么中国要搞"信创"？

**一句话**：美国不卖芯片给中国了，我们得自己造！


## 第二课：芯片到底是啥？

### 2.1 CPU vs GPU vs NPU

| 芯片类型 | 像什么 | 擅长做什么 | 代表产品 |
|----------|--------|------------|----------|
| **CPU** | 大学教授 | 什么都能做，但一次只做一件事 | Intel, 海光, 龙芯 |
| **GPU** | 1000个小学生 | 只会做简单加法，但1000个人同时做 | Nvidia, 摩尔线程 |
| **NPU** | 专门的AI工人 | 只会做AI计算，但做得特别快 | 华为昇腾, 寒武纪 |

### 2.2 为什么AI需要GPU？

**深度学习 = 大量矩阵乘法**

```python
# 神经网络的本质
output = 权重矩阵 × 输入向量 + 偏置

# 一个小模型可能有：
# - 1000万个参数
# - 每秒做1000亿次乘法
```

**CPU**: 一个教授做1000亿次乘法 → 太慢！
**GPU**: 1000个小学生，每人做1亿次 → 快多了！

### 2.3 Nvidia为什么这么牛？

不只是因为硬件好，更因为**软件生态**！

```
┌─────────────────────────────────────────────────────────┐
│                    Nvidia的护城河                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  CUDA = Nvidia发明的GPU编程语言                           │
│                                                         │
│  就像：                                                  │
│  - 英语是世界通用语言                                     │
│  - CUDA是AI世界的通用语言                                 │
│                                                         │
│  所有AI框架 (PyTorch, TensorFlow) 都用CUDA写的            │
│  20年积累，400万开发者，3000+优化库                        │
│                                                         │
│  国产芯片的难点：                                         │
│  硬件可以追，软件生态追不上！                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**你的机会在这里**：帮国产芯片补软件生态！

### 3.1 CPU公司（造电脑/服务器"大脑"的）

## 第四课：什么是"软件生态"？为什么重要？

### 4.1 一个生活中的例子

**iPhone为什么比其他手机好用？**

不只是硬件好，而是：
- App Store有几百万个App
- 开发者都愿意给iPhone写软件
- 用户体验一致

这就是**生态**！

### 4.2 AI芯片的软件生态

```
你写的PyTorch代码
        ↓
    PyTorch框架
        ↓
      CUDA (Nvidia的语言)
        ↓
     Nvidia GPU
```

**问题来了**：如果换成国产芯片呢？

```
你写的PyTorch代码
        ↓
    PyTorch框架
        ↓
      ???  ← 这里断了！国产芯片不认识CUDA！
        ↓
     国产GPU
```

### 4.3 国产芯片怎么解决这个问题？

**方案1：自己做一套（华为路线）**
```
PyTorch/MindSpore
        ↓
      CANN (华为自己的语言)
        ↓
     昇腾NPU
```

**方案2：模仿CUDA（海光路线）**
```
PyTorch代码
        ↓
      类CUDA (几乎一样)
        ↓
     海光DCU
```

**方案3：翻译器（寒武纪路线）**
```
CUDA代码 → 启明XPiler翻译器 → 寒武纪代码
   ↓                              ↓
Nvidia                          寒武纪MLU
```

### 4.4 为什么这是你的机会？

| 工作内容 | 人才供给 | 需求 | 结论 |
|----------|----------|------|------|
| 训练模型调参数 | 非常多 | 饱和 | 内卷 |
| CUDA→国产芯片迁移 | **极少** | 爆发 | **蓝海** |
| AI芯片软件开发 | **极少** | 急需 | **蓝海** |

**你的优势**：懂算法 + 有硬件基础 = 稀缺人才！

## 第八课：常见问题解答

### Q1: 我没有芯片相关经验，能转吗？

**可以！** 芯片公司招的是"懂算法+愿意学底层"的人，不是要你已经会。

**你的竞争优势**：
- 算法功底扎实
- 有硬件基础（你说你有）
- 愿意学习新东西

### Q2: AI芯片软件会不会很快被AI替代？

**不会！** 原因：

1. **需要深度理解硬件**：AI写代码可以，但不懂硬件
2. **需要跨领域知识**：算法+系统+硬件，AI很难整合
3. **需要解决新问题**：每款新芯片都有新挑战
4. **最核心的系统层**：越底层越难替代

### Q3: 应该去大厂还是芯片公司？

| 选择 | 优点 | 缺点 | 适合谁 |
|------|------|------|--------|
| 大厂 | 品牌好、培训好、薪资高 | 螺丝钉、内卷 | 想要安稳的 |
| 芯片公司 | 成长快、稀缺性、有意义 | 可能辛苦 | 想要差异化的 |

**我的建议**：如果你想弯道超车，选芯片公司。

### Q4: 具体应该投哪些公司？

**第一梯队** (最推荐)：
- 华为昇腾 - 生态最大
- 寒武纪 - 上市公司
- 海光信息 - 最像CUDA

**第二梯队** (也很好)：
- 百度昆仑芯
- 燧原科技
- 摩尔线程

**大厂ML团队** (备选)：
- 字节AML
- 阿里PAI
- 腾讯优图
